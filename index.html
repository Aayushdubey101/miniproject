Contact Book

A Mini Python Project by Aayush Dubey

About the Project

The Contact Book is a Python-based project designed to help users manage their contacts efficiently. It features functionalities like adding, deleting, updating, and searching contacts, all through a simple and user-friendly interface.

Features

  • Add new contacts
  • Delete existing contacts
  • Update contact details
  • Search for specific contacts
  • View all saved contacts

How to Run

  1. Clone the project repository from GitHub.
  2. Ensure Python is installed on your system.
  3. Run the python filename.py command to start the application.
